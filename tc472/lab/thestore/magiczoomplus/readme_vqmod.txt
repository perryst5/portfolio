INSTALLATION:

If you use VQmod and if installation using web installer failed, please continue using these instructions below:

1. Copy 'vqmod', 'catalog' and 'admin' folders to your OpenCart directory (on your server), keeping the file and folder structure intact.

2. Open your OpenCart admin panel. Go to: [Extensions > Modules > Magic Zoom Plus > Click 'Install'].

3. Enable Magic Zoom Plus™ [Extensions > Modules > Magic Zoom Plus > Edit > Enable/Disable module > Select 'Enable'].

4. The Magic Zoom Plus is now installed!

5. Customize the Magic Zoom Plus™ module through the [Extensions > Modules > Magic Zoom Plus > Edit] menu in your OpenCart admin panel.

6. To upgrade your version of Magic Zoom Plus (which removes the "Please upgrade" text), buy Magic Zoom Plus and overwrite the catalog/view/javascript/magiczoomplus.js file file with the new one in your licensed version.
