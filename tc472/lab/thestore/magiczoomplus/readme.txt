INSTALLATION:

NOTE: To manually install Magic Zoom Plus please read 'readme_manual.txt'.

1. Unzip the file on your computer and FTP the magiczoomplus folder to your OpenCart directory (on your server), keeping the file and folder structure intact.

2. Go to http://www.yoursite.url/magiczoomplus/ in your browser > click 'Install' (replace yoursite.url with your domain name).

3. Go to [Admin panel > Extensions > Modules] and install module.

4. Remain on [Admin panel > Extensions > Modules] and click 'Edit' by the Magic Zoom Plus listing and enable module.

5. The demo version is now installed :)

6. Customize the Magic Zoom Plus module through the [Extensions > Modules > Edit] menu in your OpenCart admin panel.

7. To upgrade your version of Magic Zoom Plus (which removes the "Please upgrade" text), buy Magic Zoom Plus and overwrite the catalog/view/javascript/magiczoomplus.js file file with the new one in your licensed version.
